經驗值曲線解析

20 億經驗值不是編出來的，是複利效果

首頁 ／ 攻略文章 ／ 經驗值曲線解析

經驗值表為什麼後期這麼陡？新楓之谷經典版

最後更新：2026/07/12

把目前等級跟目標等級填進練等計算機，很多人第一次看到 Lv.199 升 Lv.200 要將近 20 億經驗值都會愣一下——這不是打錯數字，而是舊版楓之谷經驗值表本身的設計。這篇用實際的經驗值表數字，拆解這條曲線後期為什麼長這樣。

先看實際數字

下面是本站經驗值表裡幾個等級「升下一級」所需的經驗值，取自舊楓之谷（pre-Big Bang）流傳已久的版本：

等級升下一級所需經驗值
Lv.1 → 215
Lv.10 → 111,716
Lv.30 → 3195,700（9.6 萬）
Lv.50 → 51709,716（71 萬）
Lv.100 → 10110,223,168（1022 萬）
Lv.150 → 151147,262,175（1.47 億）
Lv.199 → 2002,011,069,705（20.11 億）

從 15 點到 20 億，中間差了超過一億倍。但如果只看「每一級比前一級多要求多少百分比」，會發現完全不同的畫面。

關鍵：50 級之後，每級都固定多 5.5%

把每一級所需經驗值拿來跟前一級比，算出成長比例：

等級比前一級多要求
Lv.10 → 11多 38.2%
Lv.30 → 31多 14.3%
Lv.50 → 51多 8.3%
Lv.70 → 71多 5.5%
Lv.100 → 101多 5.5%
Lv.120 → 121多 5.5%
Lv.150 → 151多 5.5%
Lv.199 → 200多 5.5%

從約 Lv.50 開始，這個成長比例幾乎精準鎖定在 5.5%，一路維持到 Lv.200 都沒有明顯變化。單看 5.5% 感覺不大，但這是「複利」——每一級都是在前一級已經放大過的基礎上再乘上 1.055，連續乘 100 多次之後，數字會被滾雪球式地推得非常大。這跟銀行定存利率算複利是同一個數學原理，只是這裡的「利率」高很多、「年數」也多很多。

前期（Lv.1~50）反而不規則

從上面的表可以看到，Lv.10→11 是多 38.2%，Lv.30→31 掉到 14.3%，Lv.50→51 又降到 8.3%——前期的成長比例並不是固定的，一路在下降、直到 Lv.50 附近才收斂成穩定的 5.5%。這比較像是遊戲設計者刻意調整過新手階段的節奏（早期升級快、有成就感），而不是套一個公式硬算出來的，後期則交給一個穩定的複利公式接手。

對規劃練等有什麼幫助

  • 50 級之前經驗值需求的成長沒有規律，練等節奏會忽快忽慢，不用太在意「怎麼這幾級特別快/特別慢」。
  • 50 級之後每級穩定多 5.5%，代表越到後期，同樣的加倍卷或效率提升，省下的「絕對經驗值」越可觀——這也是為什麼多數玩家會把加倍卷留到練等後期再集中使用。
  • 規劃長期練等進度時，與其死記每一級的數字，不如記住「50 級後大約每級多 5.5%」這個規律，抓大概的量級就好，實際數字直接交給練等計算機算。

本文數字取自本站採用的舊版楓之谷（pre-Big Bang）經驗值表，跟海外「MapleStory Classic Worlds」官方頁面比對 Lv.1~49 一致，但台版新楓之谷經典版尚未正式上線，實際數值仍可能與官方公告或封測結果有些微差異，正式資料出來後會校正並同步更新這篇文章的數字。
